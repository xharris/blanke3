What is this project?

It's a small piece of software I plan on using to develop 2D games.

Why?

It'll have all the features I need: +WYSIWYG tile and object placement +Organizes spritesheets and other resources +Code can be edited in third-party editor of choice +Exports to multiple platforms: Windows, Mac, Linux, Web, Android, iOS (all of the OSes that matter) +Editor will work on Windows with future support for Mac and then Linux

When should this project be done?

Whenever. I'm aiming for a usable build by August 2016.

I want to download and run this. How?

It runs on Node.js. I'll release an offically built version in the future, but for the moment you'll just have to figure out how node.js works.

Will this stay open source?

Probably.
